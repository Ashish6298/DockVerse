# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0-rc1] - 2026-07-10
### Added
- **Phase 1-6 Core Manager Modules**: Unified dashboard view, Container life cycles (run/stop/inspect), Image registers (pull/remove), Network bridges, and Volume mounts.
- **Phase 7-8 Studio Builders**: Interactive Dockerfile studio editor and Compose YAML live configurations parser.
- **Phase 9 Monitoring Center**: SVG real-time charts rendering CPU, memory, and networking usage.
- **Phase 10 Registry Explorer**: Remote Docker Hub/private registry credentials connection manager.
- **Phase 11-12 Swarm Manager**: Multi-node Swarm initialization, Swarm services, and stack deployments.
- **Phase 13 Secrets & Configs**: Secure Docker Config and Secret key-value stores.
- **Phase 14 Stack Manager**: Portainer-style Swarm Stack deployment.
- **Phase 15 Backup & Disaster Recovery**: Tarball volume archiving, imports, and scheduled retention policies.
- **Phase 16 Security Center**: CIS benchmark security check engine and container hardening reports.
- **Phase 17 Event Center**: Normalized Docker engine events listener and JSON/CSV audits exporter.
- **Phase 18 Policy & Compliance**: Custom governance rule builder (e.g. restrict privileged flag).
- **Phase 19 Remote Hosts Manager**: Multi-cluster connection manager for TLS/SSH Docker daemons.
