4. Architecture & Layer Verification
------------------------------------
- **Clean Architecture**: Decoupled domain boundaries. Services contain all business logic. Controllers only map HTTP envelopes.
- **State Management**: TanStack Query manages all remote states; Zustand remains restricted to UI layout preferences only.
- **Socket/WebSocket Purge**: standard REST communication only.
